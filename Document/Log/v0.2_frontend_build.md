# 项目日志 - v0.2

**日期：** 2026-01-26  
**任务：** 前端开发（Apple 风格网站）  
**状态：** 已完成

## 开发内容
- 基于 `PinS.pen` 设计完成完整前端。
- 技术栈：HTML5、原生 JavaScript、Tailwind CSS。
- 实现 Apple 风格视觉（排版、玻璃拟态、动效）。

## 实现细节

### 1. 项目结构
- 在 `package.json` 与 `tailwind.config.js` 中配置自定义色板（深海军蓝 #0A1628、强调青 #22D3EE）及 Inter 字体。
- 使用 Tailwind CLI 搭建 CSS 构建流程。

### 2. 落地页 (`index.html`)
- **导航：** 粘顶栏，背景模糊与激活状态。
- **Hero 区块：** 全屏渐变背景、视频占位、大号排版（80px 标题）、滚动提示。
- **亮点区块：** 横向滚动药丸标签（效率、零碳等），动态内容切换。
- **视差区块：**
  - **安全：** 深色主题，左文右图。
  - **可持续：** 浅色主题，右文左图。
  - **创新：** 深色主题 SMR 可视化。
- **深入了解：** 交互式特性列表（手风琴风格），关联反应堆视觉占位。
- **AI 对话：** 毛玻璃容器，模拟聊天功能（用户/AI 气泡、输入中提示）。
- **页脚：** 5 列布局导航链接。

### 3. 管理后台 (`admin.html`)
- 侧边栏导航，悬停状态。
- 概览统计卡片（浏览、会话、查询）。
- 内容管理列表，状态标签（已发布/草稿）。

### 4. 交互 (`js/`)
- **滚动动效：** `IntersectionObserver` 触发区块 fade-in-up 动画。
- **标签与手风琴：** `main.js` 中 Highlights 标签与 Closer Look 特性的自定义逻辑。
- **AI 对话模拟：** `ai-chat.js` 处理消息追加与模拟输入延迟。

## 设计对齐
- 与 `PinS.pen` 设计保持一致：
  - 字体：Inter。
  - 圆角：卡片 24px/32px。
  - 颜色：严格遵循既定色板。
  - 间距：大 padding（120px）营造电影感。
